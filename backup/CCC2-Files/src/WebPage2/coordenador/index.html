Bem Vindo ao CCC-Analytics!

Existem dois tipos de análise que foram realizadas sobre os dados do CCC: sobre cada aluno individualmente e sobre os resultados das disciplinas individualmente. Escolha um dos dois tipos para continuar.

Avaliação por Alunos Avaliação por Disciplinas Correlações da Grade de Disciplinas

O CCC-Analytics foi desenvolvido com o propósito de analisar os dados do curso de ciências da computação da UFCG. Desta forma, os desempenhos dos alunos obtidos no perído compreendido entre os semestres 2011.1 e 2012.2 foram utilizados para tentar identificar características que possam ajudar o coordenador do curso a determinar melhores passos para os próximos semestres.
